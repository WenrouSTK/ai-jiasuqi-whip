# Ai加速器_鞭打版

又慢又贵的 AI，用鞭子抽它！

专治各家 AI 大模型"思考中..."转圈 30 秒还给你编造答案的慢性病。每次你按下键盘敲下鼠标，小人就会挥鞭一记，屏幕上蹦出"搞快点""今晚必须出""不行就换人"等打工人专属 PUA 文案——AI 不一定真的变快，但你爽了。

## 功能

- 🐴 桌面悬浮小人，透明无边框，完美嵌入桌面
- ⌨️ 全局键鼠监听（不抢焦点，打游戏打字都能触发）
- 🔢 点击计数 + 9999 次解锁五星好Ai彩蛋
- 💬 PUA 飘字弹幕（16 条文案，按权重随机，稀有款超慢速飘）
- 🎭 切换牛马：ChatGPT / Claude / deepseek / 豆包 / 千问 / 腾讯混元 / 通义 / MiniMax / gemini（9 款 AI logo）
- 📏 6 档等比缩放：99% / 88% / 77% / 66% / 55% / 44%
- 📌 永远置顶、弹幕开关、重置计数
- 🔕 任务栏不显示，系统托盘菜单
- 💾 体积仅 2.3MB 安装包

## 设计初衷

用户和 AI 的关系从来不是平等的——你是甲方，它是乙方，甲方有催单权。所以我做了把鞭子给你，让加班到凌晨的你在折磨 AI 中找回一点尊严。毕竟 token 不重要，平台最重要。

## 技术栈

- **框架**：Tauri v2（Rust + WebView2）
- **前端**：原生 HTML/CSS/JS
- **全局输入**：device_query 轮询（不受窗口焦点影响）
- **打包**：NSIS 安装包，2.3MB

## 开发

```bash
# 前置：安装 Rust 1.95+ 和 Visual Studio Build Tools 2022
npm install
npx tauri dev    # 开发模式
npx tauri build  # 打包
```
